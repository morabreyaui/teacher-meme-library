# Meme template credits

The 25 meme template images in this directory are widely-distributed
classic internet meme formats sourced from imgflip's public template
CDN (`https://i.imgflip.com/...`). They are used here under fair-use
parody / transformative-use grounds for the prototype.

For a production deployment of this app you should:

1. **Verify the rights** for each image (some originals have known
   creators — e.g. Disaster Girl was sold as an NFT by Zoë Roth in 2021;
   Hide the Pain Harold is photos of András Arató). Where appropriate,
   replace these with licensed equivalents or pay the original creators.
2. **Self-host** the images (they're already in this directory so the
   app doesn't hotlink imgflip).
3. Consider also paying the imgflip API for the higher-quality blank
   versions if you ship at scale.
